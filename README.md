# master-web-craft

A complete web design and engineering skill for Claude Code — built from the best open-source skills in the community.

## What it covers

| Mode | Activates when you say... |
|------|--------------------------|
| **Build** | "build a page", "create a component", "design a UI" |
| **Motion** | "animate", "add a transition", "micro-interaction" |
| **Audit** | "audit this", "what's wrong with", "prospect report" |
| **Improve** | "fix", "improve", "lighthouse", "performance", "responsive" |
| **3D** | "Three.js", "WebGL", "GSAP", "scroll-driven", "3D scene" |

## Install

```bash
npx skills add andrepereira/master-web-craft
```

Or clone and install locally:

```bash
git clone https://github.com/andrepereira/master-web-craft
npx skills add ./master-web-craft
```

## Built on

This skill synthesises techniques from:

- [taste-skill](https://github.com/Leonxlnx/taste-skill) — design intentionality
- [design-motion-principles](https://github.com/kylezantos/design-motion-principles) — motion framework
- [webdesign-agency-skills](https://github.com/peterhadorn/webdesign-agency-skills) — audit and improve loops
- [claudedesignskills](https://github.com/freshtechbro/claudedesignskills) — 3D and WebGL
- [anthropics/skills](https://github.com/anthropics/skills) — frontend quality
- [vercel-labs/agent-skills](https://github.com/vercel-labs/agent-skills) — web design guidelines
- [jeffallan/claude-skills](https://github.com/jeffallan/claude-skills) — full-stack patterns
- [jezweb/claude-skills](https://github.com/jezweb/claude-skills) — modern stack patterns
